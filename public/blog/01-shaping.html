Chapter 01

造型函数 Shaping Functions

造型函数是 shader 中最基础的工具。通过 step()、smoothstep()、pow()、exp() 等数学函数， 我们可以在 UV 坐标上绘制曲线、控制渐变，并创造出各种有趣的视觉效果。

📚 参考：The Book of Shaders 第五章

线性函数 y = x

静态
Fragment Shader

                

            

            

                最简单的造型函数：y = x。背景用 vUv.x 作为灰度值，产生从黑到白的渐变。
                绿色线条通过 step(0.01, abs(vUv.y - vUv.x)) 绘制，本质上是一个宽度为 0.01 的等值线。
            

        


        
        

            

                
指数函数 y = x⁵

                静态
            

            

                

                

                    
Fragment Shader

                    

                

            

            

                pow(x, 5.0) 产生一条陡峭的指数曲线。指数越大，曲线越"贴地"，
                只有在 x 接近 1.0 时才急剧上升。改变指数可以控制渐变的"硬度"。
            

        


        
        

            

                
自然指数 exp(x)

                静态
            

            

                

                

                    
Fragment Shader

                    

                

            

            

                exp(x) 的增长比 pow(x, n) 更自然。这里用
                (exp(x) - 1) / (exp(1) - 1) 将值归一化到 [0,1]。
                注意绿色线条用两个 smoothstep 相减来控制线条宽度。
            

        


        
        

            

                
阶跃函数 step()

                静态
            

            

                

                

                    
Fragment Shader

                    

                

            

            

                step(edge, x) 是最简单的"开关"函数：当 x < edge 返回 0.0，否则返回 1.0。
                画面被一分为二——黑与白的硬边界。这是构建更复杂图案的基础砖块。
            

        


        
        

            

                
smoothstep 绘制线条

                静态
            

            

                

                

                    
Fragment Shader

                    

                

            

            

                两个 smoothstep() 相减可以创建一条有宽度的柔和线条（"band"）。
                第一个 smoothstep 产生一个从 0 到 1 的过渡区域，第二个产生更窄的过渡区域，
                两者相减就只保留了中间的"带状"区域。
